SFH-SGP_VALIDATION_STUDY_01 — NON-PARAMETRIC AUC ANALYSIS
============================================================
INTERPRETATION BLOCK (STRICT)
============================================================

DATA: 10 subjects from OpenNeuro ds000114 motor-task dataset
K-VALUES USED: [2, 4, 8, 16, 32, 64]

METHOD: Non-parametric AUC (no sigmoid fitting)

RESULTS:
- Task AUC: -58.85 ± 1.27
- Rest AUC: -59.75 ± 0.16  
- Difference: +0.91
- t = 2.28, p = 0.049
- Cohen's d = 0.72 (medium effect)
- Bootstrap 95% CI: [0.23, 1.68]

ROBUSTNESS CHECKS:
- Bootstrap: Estimate stable across resampling
- Leave-one-out: 7/10 subjects support effect

CONCLUSION:
The AUC metric shows a statistically significant difference
between task and rest conditions (p = 0.049).

The effect size is medium (Cohen's d = 0.72), smaller than the
parametric amplitude analysis but still robust.

This confirms that residual dimensionality structure differs between
task and rest, independent of sigmoid parameterization.

NOTE: This analysis does NOT address cognitive interpretation.
It only establishes that the raw residual profiles differ.
