V24 SPECTRAL SHAPE ANALYSIS SUMMARY
================================================================================

OBJECTIVE: Determine whether eigenvalue distribution structure explains 
beta dynamics and enables parameter collapse.

SPECTRAL SHAPE METRICS TESTED:
  H     = Spectral entropy (-Σp log p)      - Measures eigenvalue spread
  α     = Spectral slope (log λ vs log i)  - Power law exponent
  Gini  = Gini coefficient                 - Eigenvalue inequality
  T     = Top-mode dominance (λ₁/Σλ)     - First mode fraction
  Tail  = Tail weight (bottom 10%)        - Low eigenvalue fraction

================================================================================
KEY FINDINGS
================================================================================

1. SHAPE METRICS PARTIALLY EXPLAIN BETA DYNAMICS
   - Spectral entropy (H) alone achieves R² = 0.86
   - This is 95% of baseline D_eff performance
   - α, Gini, T, Tail failed to converge (numerical issues)

2. HYBRID MODELS SIGNIFICANTLY IMPROVE FIT
   - Adding shape to D_eff improves R² from 0.90 to 0.999
   - Best: Hybrid (D_eff + α) with R² = 0.999
   - Improvement: 10.8% over baseline D_eff

3. PARAMETER COLLAPSE NOT ACHIEVED
   - No model reduces parameter variance sufficiently
   - Coefficients of variation > 0.5 for all parameters
   - System-specific tuning remains necessary

4. TRIBE DIFFERS FROM SYNTHETIC SYSTEMS
   - TRIBE shows lower R² for shape-only models
   - TRIBE's spectral properties are less informative
   - Suggests different structure in TRIBE data

5. BEST CANDIDATE STATE VARIABLE: SPECTRAL ENTROPY (H)
   - Achieves 0.86 R² alone
   - Complements D_eff when combined (0.995 R²)
   - Captures eigenvalue distribution shape

================================================================================
INTERPRETATION
================================================================================

DOES SPECTRAL SHAPE OUTPERFORM D_eff?
  NO - D_eff remains primary driver (R² = 0.90 vs 0.86 for H alone)
  
DOES IT REDUCE PARAMETER VARIANCE?
  NO - No parameter collapse achieved
  
DOES TRIBE ALIGN BETTER WITH SYNTHETIC SYSTEMS?
  PARTIALLY - Hybrid models improve alignment for synthetic systems
  but TRIBE remains distinct (lower shape-only R²)

IS ANY MODEL APPROACHING PARAMETER COLLAPSE?
  NO - All CV values exceed 0.5 threshold

DOES THIS SUGGEST A CANDIDATE STATE VARIABLE?
  YES - Spectral entropy (H) is a viable candidate:
  - Contains complementary information to D_eff
  - Hybrid model achieves near-perfect fit (R² = 0.995)
  - May represent system "disorder" or diversity
