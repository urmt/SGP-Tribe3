# REPLICATION TIERS
## Evidence Level Definitions

---

## TIER 0: EXPLORATORY

**Requirements:**
- Single observation
- No controls required
- No claim validation

**Evidence Label:** PROVISIONAL

**Example:**
- First observation of a phenomenon
- Pilot analysis
- Hypothesis-generating

**Limits:**
- Cannot be cited as evidence
- Cannot support robustness claims
- Must be followed by T1+ analysis

---

## TIER 1: SINGLE-CONTROL

**Requirements:**
- Single control tested
- Survival or destruction documented

**Evidence Label:** OBSERVED

**Example:**
- Phase 177 (phase random only)

**Limits:**
- Cannot claim robustness
- Indicates need for more controls

---

## TIER 2: MULTI-CONTROL

**Requirements:**
- 3+ controls tested
- Mixed survival/destruction pattern

**Evidence Label:** REPLICATED or CONTROL_DEPENDENT

**Example:**
- Phase 179 (5 controls, mixed results)

**Limits:**
- Can support moderated claims
- Requires interpretation of which controls affect result

---

## TIER 3: SURROGATE-RESISTANT

**Requirements:**
- 5+ controls tested
- Most/all survive

**Evidence Label:** SURROGATE_RESISTANT

**Example:**
- Hypothetical phase where all controls fail to destroy finding

**Limits:**
- Strong evidence
- Still requires caution on interpretation

---

## TIER 4: REPLICATION-READY

**Requirements:**
- Full control set (6 controls)
- Strict LOSO validation
- Parameter registry compliance

**Evidence Label:** REPLICATED with full documentation

**Example:**
- Not yet achieved in current pipeline

**Limits:**
- Ready for independent verification

---

## TIER 5: PUBLICATION-GRADE

**Requirements:**
- Tier 4 + independent verification
- Full audit chain
- All controls documented

**Evidence Label:** VERIFIED

**Example:**
- Not yet achieved

**Limits:**
- Highest confidence level
