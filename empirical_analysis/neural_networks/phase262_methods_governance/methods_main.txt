1. KURAMOTO MODEL (TIER 1 - Validated Core)
================================================================================

Definition:
  The Kuramoto model describes N coupled phase oscillators with natural
  frequencies omega_i and coupling strength K:

    d_theta_i/dt = omega_i + (K/N) * sum_{j=1}^N sin(theta_j - theta_i)

Implementation used in Phases 252-259:
  - N: 16, 32, 64, 128 (varying by phase)
  - Coupling K: 0-4 or 0-6 (varied per phase)
  - Frequency distributions: Gaussian (mu=0, sigma=1) OR Uniform (omega ~ U[-2,2])
  - Topology: All-to-all (dominant), Erdős-Rényi p=0.25 (Phase 253)
  - Integration: Euler, dt=0.01 or 0.05
  - Time steps: 300-2000 (varies by phase)

Provenance: phase252_minimal_transition_test.py, phase252_finite_size_test.py,
            phase252_script8.py (verified in phase258_filesystem_verification.txt)

2. ORDER PARAMETER R (TIER 1 - Validated Core)
================================================================================

Definition:
  The Kuramoto order parameter measures global phase synchronization:

    R * e^(i * psi) = (1/N) * sum_{j=1}^N e^(i * theta_j)

  where 0 <= R <= 1:
    R = 0  -> fully disordered (random phases)
    R = 1  -> fully synchronized (identical phases)

Implementation:
  R = |sum_{j=1}^N e^(i * theta_j)| / N
  Computed as: R = sqrt((sum cos(theta_j))^2 + (sum sin(theta_j))^2) / N

Usage in phases:
  - Phase 252-1/2: R(K=0)=0.079, R(K=6)=0.983
  - Phase 254: hysteresis test measures R for forward/backward K sweeps
  - Phase 255: relaxation measured by R(t) convergence

Provenance: All phase252-255 scripts compute R as standard output.

4. SUSCEPTIBILITY ESTIMATOR (TIER 1 - Validated Core)
================================================================================

Definition:
  Susceptibility measures fluctuations in the order parameter:

    chi = N * Var(R)

  where Var(R) is the variance of R over time windows or trials.

Implementation (Phase 252-6, 252-7, 252-8):
  - Time-window variance: chi = N * (<R^2> - <R>^2)
  - Computed over STEPS=2000 time steps (Phase 252 scripts)
  - Or STEPS=1000 (Phase 256 lean version)

Results:
  - chi peak grows with N
  - gamma = 1.14 from log-log fit, R2 = 0.985 (Phase 252-6)
  - BUT: gamma varies 0.62-1.98 with frequency distribution (Phase 253)

IMPORTANT LIMITATION:
  gamma is NOT universally 1.14. The value depends on frequency distribution.
  Only A2A + Gaussian gives gamma ~1.14. Claim C2 applies only to that condition.

5. BINDER CUMULANT (TIER 1 - Validated Core)

  The Binder cumulant is defined as:

    U4 = 1 - <m^4> / (3 * <m^2>^2)

  where m = R (the order parameter). For the Ising universality class,
  U4 reaches 2/3 in the disordered phase and 0 at criticality.

6. HYSTERESIS TEST (TIER 1 - Validated Core)
================================================================================

Definition:
  Hysteresis tests whether the phase transition is continuous (no hysteresis)
  or first-order (hysteresis present). Forward and backward K sweeps measure
  loop_area = integral(|R_forward(K) - R_backward(K)| dK).

Implementation (Phase 254):
  - Forward sweep: K from 0 to 4, initialize from random theta0
  - Backward sweep: K from 4 to 0, initialize from final forward state
  - Compute: loop_area = sum(|R_f - R_b|) * delta_K

Results:
  - loop_area = 0.0304 << 0.15 threshold
  - max_gap = 0.0338
  - VERDICT: Continuous (non-hysteretic)

CRITICAL FAILURE (Documented in Failure Registry):
  - Backward sweep initializes from forward end-state (coupled)
  - Not truly independent initialization
  - loop_area may be UNDER-ESTIMATED

Provenance: phase254_script10.py
            (verified in phase258_filesystem_verification.txt)

7. CRITICAL SLOWING ESTIMATOR (TIER 1 - Validated Core)
================================================================================

Definition:
  Critical slowing measures the relaxation time tau near the critical point.
  At Kc, fluctuations persist longer, and tau increases.

Implementation (Phase 255):
  - Define R_final = mean(R[STEPS-100:STEPS])
  - Find tau = first step where |R(t) - R_final| < 0.1 * (R_final - R_disordered)
  - Measure tau(K) across K range
  - K_peak = K where tau is maximum

Results:
  - K_peak = 1.4222 (tau maximum)
  - peak_ratio = 5.89 (5.89x increase at Kc relative to baseline)
  - tau_peak = 50.0 (MAXIMUM VALUE - SATURATED)

IMPORTANT FAILURE (Documented in Failure Registry):
  - tau reaches maximum (50.0) for all N=(16,32,64)
  - 50-step stability criterion not satisfied within STEPS=1000
  - tau cannot reliably identify Kc; all tau_peak values identical
