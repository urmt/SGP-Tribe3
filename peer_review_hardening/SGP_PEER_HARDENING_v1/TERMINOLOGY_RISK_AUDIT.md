# TERMINOLOGY RISK AUDIT

| High-Risk Term | Alternative / Downgrade | Rationale |
| :--- | :--- | :--- |
| **Universal** | Consistent | "Universal" implies a law; "consistent" describes the observation. |
| **Invariant** | Stable parameter | Avoid claiming "invariance" without formal proof. |
| **Organizational Law** | Scaling pattern | "Law" is premature; "pattern" is descriptive. |
| **Semantic Compression** | Data dimensionality reduction | "Semantic" is highly subjective; "data" is precise. |
| **Attractor** | Metastable state | "Attractor" has rigid dynamical meaning; "metastable state" is safer. |
| **Topological Signal** | Manifold structure | "Signal" is vague; "structure" is grounded. |
| **Geometric Truth** | Empirical observation | "Truth" is not a scientific term in this context. |
